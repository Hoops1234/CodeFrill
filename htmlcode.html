HTML

What Is HTML?

HTML stands for HyperText Markup Language. It’s not a programming language that makes things move or calculate.
It’s a markup language, which means it tells the computer how to organize and display content on a webpage.
Think of HTML like the blueprint for a webpage. It doesn’t make things pretty or interactive, it builds the basic structure:

headings, paragraphs, images, links, and more.

How HTML Works

HTML uses tags to mark different parts of a webpage. Tags are written inside angle
brackets like "tagname". Most tags come in pairs:

  • An opening tag: "p"
  • A closing tag: "/p"

Everything between those tags is the content that will be shown.



!DOCTYPE HTML

This tells the browser that the page uses HTML5, which is the latest version of HTML.

You only need this once at the very top of your file

HTML and /HTML

This wraps the entire HTML document. Everything goes inside these tags.

All your content goes here

head and /head

This section contains information about the webpage, not what people see.
It can include the title, links to stylesheets, and other metadata.

For Example is
head
"title" My Website "/title"

title and /title

This sets the name of the webpage that appears in the browser tab.

body and /body

This is where all the visible content goes headings, paragraphs, images, links, etc.

p and /p

This tag is for paragraphs regular blocks of text.

a and /a

This tag creates a link to another page or website.
You need to add an href attribute to tell it where to go.

You will use it when you insert a link

img

This tag shows an image. It doesn't need a closing tag,
but it does need a src (source) and alt (alternative text).

ul, ol, and li

These tags create lists.

  • ul is for unordered lists (bullets)
  • ol is for ordered lists (numbers)
  • li is for each list item

Additional HTML Elements

div and span

div is a block-level container used to group elements together. It creates a new line before and after it.

span is an inline container used to group text or elements within a block. It doesn't create new lines.

Example: <div>This is a block container</div>

Tables

HTML tables use these tags:

  • table - wraps the entire table
  • tr - defines a table row
  • th - defines a table header cell
  • td - defines a table data cell

Example:

<table>
  <tr>
    <th>Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>John</td>
    <td>25</td>
  </tr>
</table>

Forms

Forms collect user input. Key form tags:

  • form - wraps all form elements
  • input - creates different input fields (text, password, checkbox, radio, etc.)
  • textarea - for multi-line text input
  • label - describes a form element
  • button - creates clickable buttons
  • select and option - create dropdown menus

Example:

<form>
  <label for="username">Username:</label>
  <input type="text" id="username" name="username">
  <input type="submit" value="Submit">
</form>

Common Input Types

  • type="text" - single-line text input
  • type="password" - masked text input
  • type="email" - email address input
  • type="number" - numeric input
  • type="checkbox" - checkbox for multiple choices
  • type="radio" - radio button for single choice
  • type="submit" - submit button

Semantic HTML5 Elements

Semantic elements describe their purpose in the code:

  • header - introductory content or navigation
  • nav - navigation links
  • main - main content of the page
  • section - thematic grouping of content
  • article - self-contained content
  • aside - content related to the main content
  • footer - footer for a section or page

Attributes

Attributes provide additional information about elements:

  • class - identifies elements for styling (multiple elements can share a class)
  • id - unique identifier for one element (used for anchors and JavaScript)
  • src - specifies the source URL for media (images, videos)
  • href - specifies the destination URL for links
  • alt - alternative text for images (accessibility and SEO)
  • style - inline CSS styling

HTML Comments

Comments are notes in the code that don't appear in the browser:

<!-- This is an HTML comment -->
<!-- Comments can span
     multiple lines -->

Escape Characters (HTML Entities)

Use these when you need special characters:

  • &lt; = < (less than)
  • &gt; = > (greater than)
  • &amp; = & (ampersand)
  • &quot; = " (quotation mark)
  • &nbsp; = non-breaking space

Meta Tags

Metadata provides information about the document:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1">
<meta name="description" content="Page description for SEO">

Best Practices

  • Always indent nested elements for readability
  • Use semantic elements whenever possible
  • Include alt attributes on all images
  • Use lowercase for tag and attribute names
  • Close all tags properly
  • Validate your HTML (use W3C validator)
